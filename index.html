Что я использую в работе
HTML

Основа для создания внешнего облика интернет-сайтов и веб-приложений. Язык разметки HTML версии 5 позволяет создавать не только удобный и красивый дезайн, но и оптимизировать Ваш сайт для поисковых систем.

CSS3

Этот язык стилей позволяет создавать абсолютно любой внешний вид Вашего сайта или приложения. Все ограничивается только вашей фантазией!

Java Script

Язык программирования, который позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое

Node.js

Платформа, создающая внешний вид Вашего сайта или приложения на языке Java Script.

Spring Framework

Мощная "начинка" для создания приложения любого назначения! Позволяет извлечь максимум из возможностей языка Java и решить любые бизнес-задачи.

Thymeleaf

Еще одна платформа для создания внешнего вида сайта или веб-приложения. Позволяет использовать все возможности HTML и работает в отличной связке со Spring Framework

MySql

Самая распространенная система управления базами данных для создания сайтов и приложений. Позволяет решать большинство задач с минимальными затратами и хорошей скоростью.

Bitrix24

Корпоративный портал "Битрикс24" - это отличное решение для автоматизации многих процессов деятельности любой организации. Работа с проектами, постановка задач, планирование и автоматизация бизнес-процессов - вот далеко не полный список возможностей.

Bitrix CMS

Система управления сайтом "1С:Битрикс" обеспечивает максимальные возможности для создания сайтов любой сложности "с нуля".
